Tablas de multiplicar

El proyecto es una página web interactiva que permite a los usuarios generar tablas de multiplicar de un número específico. El tipo de software es una aplicación web basada en HTML, CSS y JavaScript. El propósito de esta aplicación es brindar a los usuarios una manera sencilla y visual de generar y ver tablas de multiplicar de un número que pueden ingresar en un campo de entrada.

Tecnologías: HTML, CSS, JavaScript

Página Panther Shop

El propósito de la aplicación es ofrecer a los usuarios una plataforma en la que puedan explorar y comprar productos relacionados con deportes, como playeras, shorts, calcetas, tenis, guantes, balones y más.
